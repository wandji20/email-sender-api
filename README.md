## Email API

> Emailer for sending personal emails. <br>
> This a simple api that allows me to send emails to my email address

## Built with

- Ruby on Rails
- Sidekiq
- Redis

## Endpoint

- https://email-api-wandji.herokuapp.com/api/v1/portfolio
- Params
  > email: required <br>
  > message: required <br>
  > name: optional <br>
  > website: optional
